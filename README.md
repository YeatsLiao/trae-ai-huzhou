# 情绪天空 (Emotion Sky)

> 每个人都能点亮一片天空，每个地方都能看到一座城市的情绪天气

TRAE AI 创造力大赛参赛项目 —— 生活娱乐赛道

---

## 📋 项目简介

情绪天空是一个让大众共同参与的情绪可视化平台：
- 用户选择当前情绪，系统将其转化为一片"天气"
- 同一区域的用户情绪汇聚成城市的"情绪天空"
- 通过赛博朋克风格的城市天际线天空场景，直观展示集体情绪

**核心理念**：如果把城市的集体情绪变成天气，会是什么样子？

## ✨ 功能特色

### 个人情绪天空
- 8 种情绪选择（开心/平静/焦虑/悲伤/愤怒/迷茫/疲惫/充满活力）
- 对应 8 种天气效果（晴天/多云/小雨/大雨/雷暴/雾霾/黄昏/极光）
- 动态天空渲染 + 环境音效 + 赛博朋克城市天际线

### 城市情绪地图
- 中国城市情绪天气可视化
- 实时展示各城市的集体情绪状态
- 点击城市查看详情和情绪分布

### 情绪趋势洞察
- 个人情绪历史记录
- 区域情绪趋势曲线
- 情绪规律发现

## 📊 情绪-天气映射

| 情绪 | 天气 | 天空色彩 | 环境音 |
|------|------|----------|--------|
| 开心 | 晴天 | 暖金色 → 淡蓝 | 鸟鸣、微风 |
| 平静 | 多云 | 柔和灰蓝 | 低频白噪音 |
| 焦虑 | 小雨 | 暗灰蓝 | 雨声、远雷 |
| 悲伤 | 大雨 | 深灰紫 | 暴雨声 |
| 愤怒 | 雷暴 | 暗紫红 | 雷声、暴雨 |
| 迷茫 | 雾霾 | 浑浊灰白 | 寂静 |
| 疲惫 | 黄昏 | 橙红 → 深紫 | 虫鸣、晚风 |
| 充满活力 | 极光 | 多彩流动 | 电子氛围乐 |
