# 情绪天空 - 创意构思阶段记录

## 1. 初始灵感

### 1.1 灵感来源
- 将人类情绪转化为直观的天气现象
- "城市集体情绪"的社会感知价值
- 赛博朋克视觉美学与数据可视化的结合
- 情绪疗愈与社交共感的双重价值

### 1.2 核心概念
> 如果把城市的集体情绪变成天气，会是什么样子？今天北京是晴天（大家都很开心），上海在下雨（焦虑弥漫），广州正在打雷（愤怒爆发）。

### 1.3 赛道定位
- **主赛道**：生活娱乐 / 造点新花样
- **备选叠加**：社会公益（情绪疗愈方向）

## 2. 创意评估

### 2.1 优势分析
| 维度 | 说明 |
|------|------|
| 概念新颖性 | "情绪→天气"的隐喻直观且富有诗意 |
| 视觉冲击力 | 赛博朋克城市天空天然有视觉吸引力 |
| 情绪共鸣 | "你不是一个人"的共情感 |
| 技术可行性 | 纯前端方案，快速可实现 |
| 传播潜力 | 视觉效果好，适合社交媒体传播 |

### 2.2 风险评估
- **实用性短板**：偏体验向，需强化情绪疗愈定位
- **数据真实性**：初期依赖模拟数据
- **技术复杂度**：天空渲染和音效系统需精心打磨

---

## 3. 核心设计决策

### 3.1 产品定位选择
**最终选择**：C. 情绪陪伴/疗愈向 + A. 艺术体验装置

**理由**：
- 情绪陪伴提供实用价值和用户粘性
- 艺术体验保证视觉吸引力和评审好感
- 两者结合形成差异化竞争力

### 3.2 情绪-天气映射体系

| 情绪 | 天气 | 天空色彩 | 粒子/动效 | 环境音 |
|------|------|----------|-----------|--------|
| 开心 | 晴天 | 暖金色 → 淡蓝 | 阳光粒子，光晕浮动 | 鸟鸣、微风 |
| 平静 | 多云 | 柔和灰蓝 | 云朵缓慢漂移 | 低频白噪音 |
| 焦虑 | 小雨 | 暗灰蓝 | 雨滴粒子，水面涟漪 | 雨声、远雷 |
| 悲伤 | 大雨 | 深灰紫 | 密集雨滴，雾气弥漫 | 暴雨声 |
| 愤怒 | 雷暴 | 暗紫红 | 闪电效果，屏幕震动 | 雷声、暴雨 |
| 迷茫 | 雾霾 | 浑浊灰白 | 雾气粒子，模糊滤镜 | 寂静 |
| 疲惫 | 黄昏 | 橙红 → 深紫 | 缓慢渐变，星星隐现 | 虫鸣、晚风 |
| 充满活力 | 极光 | 多彩流动 | 极光光带舞动 | 电子氛围乐 |

### 3.3 区域聚合逻辑
- 取区域内最近 50 条情绪数据
- 按类型加权平均（开心=晴天权重最高，愤怒=雷暴权重最高）
- 输出混合天气（如"60% 开心 + 30% 平静 + 10% 焦虑"→ 晴间多云，偶有微风细雨）
- 数据不足 10 条时显示"数据积累中"

## 5. 产品形态

### 5.1 个人情绪天空
- 用户选择当前情绪，生成专属动态天空场景
- 配有对应环境音效和色彩
- 沉浸式全屏体验

### 5.2 城市情绪天气
- 同一区域用户情绪汇总为城市"情绪天气"
- 在地图上实时展示各城市情绪状态
- 点击城市查看详情和趋势

### 5.3 情绪趋势洞察
- 情绪随时间的趋势变化
- 发现"周一普遍焦虑""节假日整体开心"等规律
- 提供简单心理学解读

## 6. 创意迭代记录

### 6.1 第一版想法
- 接入 Twitter/Reddit 实时数据流
- 自动情感分析
- 天气可视化

### 6.2 优化方向（当前版本）
- 改为**用户主动提交情绪**模式（更可控、更有参与感）
- 加入**匿名社交**元素（共情感）
- 加入**情绪疗愈**功能（增强实用性）

### 6.3 未来扩展
- 接入真实社交媒体数据（作为数据源补充）
- AI 心理学解读
- 成就系统和社交分享
