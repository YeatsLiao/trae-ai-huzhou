# 情绪天空 - 一个藏着天气的心情树洞

## 1. 这个想法是怎么来的

### 1.1 说点掏心窝子的
不知道你有没有过这种时刻——
下班路上挤着地铁，心里堵得慌，却不知道该跟谁说；
加班到深夜盯着屏幕，突然就鼻子一酸，也说不清是为什么；
有时候开心得想蹦跶，翻了一圈通讯录，又觉得"这点小事至于吗"……

我们这代人好像都挺会"憋着"的。朋友圈要经营，微博有熟人，连日记写下来都怕哪天被自己翻出来尴尬。情绪这东西，看不见摸不着，堵在心里又难受。

于是就想啊——能不能有这么一个地方，像个树洞似的，你往里扔一句话、一个心情，没人知道你是谁，但你知道有人在听。

然后又想到小时候总说"今天天气好好，心情也跟着好起来了"，那反过来呢？心情不好的时候，是不是心里也在下雨？

**情绪天空**就这么来了：你的心情是什么天气，就把它放出来，飘在这座城市的上空。

### 1.2 核心概念（说人话版）
> 每个人的心里都有一片天。
> 今天你开心，就出太阳；你难过，就下点雨；你烦躁，就打个雷。
> 然后你抬头一看——哇，原来今天上海有好多人跟我一样在下雨啊，
> 原来北京的朋友们今天都挺嗨的。
> 你不是一个人。

### 1.3 这玩意儿属于啥
- **主赛道**：生活娱乐 / 就是想整点有意思的
- **隐藏属性**：一个匿名的心情树洞，谁都能来唠两句

## 2. 凭啥觉得这事儿能成

### 2.1 它好在哪（真心话）
| 点 | 说说为啥 |
|----|----------|
| 好懂 | 开心=晴天，难过=下雨，是个人就能get到，不用解释半天 |
| 好看 | 谁不喜欢看好看的东西呢？天空飘来飘去的，解压 |
| 暖心 | 最戳人的就是那句"原来不是我一个人这样"，孤独感瞬间少一半 |
| 好做 | 不用搭复杂的后端，前端就能跑出个像样的demo |
| 好分享 | 截个图发朋友圈，"今天我的天空是极光色的"，多酷 |

### 2.2 得心里有数的事儿
- **纯看的话有点虚**：光好看不够，得真的能帮到人，所以得往"陪伴感"上靠
- **一开始人少怎么办**：先用模拟数据顶着，等人多了就真实了
- **效果得用心磨**：天空好不好看、音效能不能让人静下来，这些细节很重要

## 3. 打算怎么做

### 3.1 想做成啥样
**核心定位**：一个好看又暖心的心情树洞 + 一片能看见情绪的天空

说具体点：
- 首先得**好看**——毕竟这是个看脸的时代，天空得美到让人想截图
- 然后得**暖心**——不能光好看，得让人觉得"被懂了"，孤独的时候知道有人陪着
- 两者加起来，就是我们想要的样子

### 3.2 心情和天气怎么对上号
就像我们常说的"心里下雨了"，每种心情对应一片天空：

| 你现在的心情 | 你的天空是 | 长啥样 | 听着像啥 |
|-------------|-----------|--------|----------|
| 美滋滋的 | 大晴天 | 暖金色的阳光，光晕慢悠悠飘 | 鸟叫、风吹树叶 |
| 挺平静的 | 多云天 | 灰蓝色的云，慢慢飘 | 轻轻的白噪音 |
| 有点烦 | 小雨天 | 暗灰蓝，雨滴落下来泛起涟漪 | 雨声、远处闷闷的雷 |
| 好难过 | 大雨天 | 深紫灰色，雨哗哗下，雾蒙蒙的 | 暴雨哗啦啦 |
| 气死我了 | 雷暴天 | 暗紫红色，闪电劈下来，屏幕都在抖 | 雷声、暴雨 |
| 搞不懂啊 | 雾霾天 | 浑浑的灰白色，雾蒙蒙的看不清 | 静悄悄的 |
| 累瘫了 | 黄昏天 | 橙红慢慢变深紫，星星悄悄冒出来 | 虫鸣、晚风 |
| 干劲十足 | 极光天 | 五颜六色的光带在天上跳舞 | 电子感的氛围乐 |

### 3.3 城市的天气怎么算出来的
简单说就是把大家的心情凑一块儿：
- 一个城市最近的心情凑一起算
- 开心的人多就是晴天，难过的人多就是下雨
- 还能混着来——比如大部分人开心，少数人焦虑，就是"晴间多云，偶尔飘点小雨"
- 人太少的时候就说"这片天空正在养成中～"

## 4. 谁会用这玩意儿

### 主要是这群人
大概就是18到35岁的朋友们——
- 上班上累了的打工人
- 读书读烦了的学生党
- 没事喜欢琢磨自己心情的人
- 就是单纯觉得好看想玩玩的人

### 什么时候会打开它
- 摸鱼的时候，想记录一下"我现在好烦"
- 睡觉前，回顾一下今天过得咋样
- 情绪上头的时候，找个地方发泄一下
- 单纯好奇"今天大家心情都咋样"的时候

### 为啥需要它
- **懒得写日记**：传统日记太麻烦了，谁有那闲工夫写一大段
- **想看看自己**：有时候翻翻记录，发现"哦原来我上周这么丧啊"
- **不想一个人扛**：知道有人跟自己一样，就没那么孤独了

## 5. 具体能玩啥

### 5.1 你的专属天空
选一下你现在啥心情，立马给你变出一片对应的天空
- 好看的动态效果，雨啊雪啊极光啊都有
- 配上对应的声音——下雨就听雨声，晴天就听鸟叫
- 安安静静地看一会儿，心情好像真的能平静点

### 5.2 看看大家的心情
打开地图，看看全国各地的朋友们今天都咋样
- 北京今天是大晴天？那大家心情都不错嘛
- 上海在下雨？哦，原来不止我一个人emo
- 点进去还能看看这个城市最近的心情变化

### 5.3 回头看看自己
记录得多了，就能看到自己这段时间的心情曲线
- 发现"哦我原来每周一都挺丧的"
- 或者"最近晴天好多啊，看来过得不错"
- 顺便给点小建议——比如丧的时候可以试试出去走走

## 6. 想法是怎么一步步变的

### 6.1 最开始想的是……
一开始想的是扒社交媒体的数据，自动分析大家的情绪，然后做成天气图。
后来觉得吧——
- 扒数据有点难搞
- 自动分析也不一定准
- 最重要的是，少了点"人味儿"

### 6.2 现在的想法（感觉好多了）
改成**大家主动来记录心情**，就像写日记一样，只是更简单：
- 选个心情就行，不用打字，门槛低
- 匿名的，没人知道你是谁，放心说
- 还能看到大家的心情，有陪伴感
- 做得好看点，看着也舒心

### 6.3 以后还能玩啥（画大饼时间）
如果这个demo做出来反响还不错，后面可以加的东西可多了：

- **登录功能**：有个自己的账号，心情记录能存住，换手机也不丢
- **树洞社区**：不只是选心情，还能写几句话扔进去，别人能看到能点赞，但就是不知道你是谁
- **交朋友**：哎？这个人跟我心情总是一样的，要不要打个招呼？
- **AI 陪聊**：心情不好的时候，AI 来跟你唠两句，给点小建议
- **成就徽章**：连续记录7天啦、这个月有20天是晴天啦，收集小徽章也挺有意思
- **分享功能**：把今天的天空分享到朋友圈，装个X

## 7. 关于比赛

### 7.1 这次先做好啥
- **好看是第一位的**：天空效果一定要够炫，一眼就能抓住人

### 1. 这是个啥

**名字**：情绪天空 (Emotion Sky)

**想解决啥问题**：
现在的人啊，压力都挺大的。开心了想分享怕被说矫情，难过了想吐槽又怕打扰别人，情绪堵在心里找不到出口。而且吧，很多时候我们自己都没搞懂——"我这段时间到底咋了？"

**怎么想到的**：
小时候总听大人说"今天天儿好，心情都跟着敞亮"。那反过来想——心情好不好，是不是心里也有一片天？开心就是大太阳，难过就是下雨，烦躁了还能打个雷。

然后又想啊——如果把全中国网友的心情都凑到一起，变成一座座城市的"情绪天气"，会不会挺有意思的？就像一个超大的匿名树洞，你扔一个心情进去，发现"哇，原来这么多人跟我一样啊"，瞬间就不孤独了。

**到底是个啥产品**：
一个能把心情变成天气的 Web 应用。选一下你现在啥心情，立马给你变出一片对应的天空（还带音效的那种）。还能看看全国各城市今天的"情绪天气"——北京晴没晴？上海下雨了吗？顺便还能记录自己的心情趋势，当个懒人版心情日记用。

### 2. 给谁用的，啥时候用

**主要面向谁**：
18到35岁的年轻人——不管你是天天加班的打工人，还是考试考到头秃的学生党，只要你偶尔想记录一下心情、或者单纯好奇"大家今天都咋样"，都能来玩。

**啥时候会打开它**：
- 上班摸鱼的时候，随手记一下"我现在好烦"
- 晚上睡觉前，翻翻今天自己是啥天气
- 情绪上头的时候，找个地方发泄一下（反正匿名的）
- 单纯好奇，想看看别的城市的朋友们今天心情咋样

**大家的痛点**：
- 写日记太麻烦了，坚持不下来
- 不知道自己情绪啥规律，稀里糊涂的
- 有时候觉得"就我一个人这样吗"，挺孤独的

### 3. 这玩意儿有啥好的

**对个人来说**：
记录心情比写日记快多了——点一下就完事。看多了自己的心情曲线，慢慢也能摸出点规律，比如"哦原来我每周一都丧"，也算更了解自己了吧。

**对大家来说**：
最暖心的就是那句"原来不是我一个人"。看看城市情绪地图，发现今天好多人跟自己一样在下雨，瞬间就觉得——害，大家都这样，我也不是特例。这种"被陪伴"的感觉，比啥都强。

### 4. 以后还能咋玩（画大饼）

现在这个 demo 只是个开始，以后好玩的可多了：
- **用户登录**：有自己的账号，心情记录不会丢，换手机也不怕
- **树洞社区**：不只是选心情，还能写几句话扔进去，别人能看能点赞，就是不知道你是谁
- **社交功能**：哎？这个人跟我心情总能对上，要不要认识一下？
- **AI 唠嗑**：心情不好的时候，AI 来陪你说说话，给点小建议
- **成就系统**：连续记录7天啦、这个月晴天最多啦，收集小徽章也挺有成就感的
- **分享装逼**：把今天的天空分享到朋友圈，酷得很
