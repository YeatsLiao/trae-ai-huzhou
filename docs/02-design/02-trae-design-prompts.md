## 提示词 1：首页设计（个人情绪天空）

**用途**：生成核心首页设计

```
生成一款名为"情绪天空"的情绪可视化 Web 应用的移动端高保真设计稿（375×812，深色模式），首页设计：

【页面内容】
① 顶部：极简导航栏（左上角 Logo：抽象天空图标 + "情绪天空"文字；右上角设置图标）
② 主体区域（占屏幕 70%）：全屏沉浸式赛博朋克城市天际线天空场景，展示动态天气效果（默认晴天：暖金色天空渐变 + 阳光粒子 + 霓虹城市剪影）
③ 底部区域（占屏幕 30%）：情绪选择器卡片，8 种情绪以圆形图标 + 文字形式排列成两行，每个情绪对应颜色：
   - 开心（黄色）、平静（蓝色）、焦虑（灰色）、悲伤（深紫）
   - 愤怒（红色）、迷茫（白色）、疲惫（橙色）、充满活力（青色）
   选中状态有发光效果
④ 底部导航栏：三个 Tab（个人天空 / 城市地图 / 情绪趋势），当前激活"个人天空"

【视觉调性】
- 赛博朋克 + 数据可视化美学
- 深色背景（#0a0e1a），避免纯黑
- 强调色：天蓝色(#38bdf8)、紫色(#a78bfa)、粉色(#f472b6)
- 圆润控件（圆角 12–16px）
- 充足留白，层次清晰
- 天气效果要有动态感（粒子、渐变动画）

【交互提示】
- 点击情绪图标 → 天空场景切换到对应天气效果
- 城市天际线有霓虹灯光闪烁效果
- 情绪选择器卡片有轻微浮起阴影
```

## 提示词 2：城市情绪地图页

**用途**：生成地图页设计

```
生成"情绪天空"应用的城市情绪地图页面（375×812，深色模式）：

【页面内容】
① 顶部：标题栏（"城市情绪天气"）+ 返回按钮 + 搜索框
② 主体区域：中国地图可视化，每个省份/城市用天气图标标记（晴天/多云/小雨/雷暴/雾霾等），鼠标悬停显示城市名和情绪分布百分比
③ 地图下方：当前城市详情卡片（例如"北京 · 晴间多云"），包含：
   - 天气图标 + 温度式情绪值（如"开心指数 72%"）
   - 情绪分布饼图（开心 45% / 平静 30% / 焦虑 15% / 其他 10%）
   - 简短描述文案（如"今天的北京是个温暖的晴天"）
④ 底部导航栏：三个 Tab，当前激活"城市地图"

【视觉调性】
- 延续首页的赛博朋克深色风格
- 地图用深蓝色调背景，城市用发光天气图标标记
- 卡片用半透明磨砂效果
- 动画：地图缩放过渡、天气图标呼吸效果
```

## 提示词 3：情绪趋势页

**用途**：生成趋势页设计

```
生成"情绪天空"应用的情绪趋势页面（375×812，深色模式）：

【页面内容】
① 顶部：标题栏（"情绪时间线"）+ 返回按钮
② 标签切换："我的情绪" / "城市趋势"
③ 主体区域（我的情绪）：
   - 情绪日历（最近 7 天，每天用天气图标标记情绪）
   - 情绪变化折线图（横轴：时间，纵轴：情绪指数）
④ 主体区域（城市趋势）：
   - 城市情绪排行榜（Top 5 城市，显示当前天气和情绪指数）
   - 今日情绪分布柱状图（8 种情绪的比例）
⑤ 底部：情绪小贴士卡片（如"今天很多人和你一样感到平静"）
⑥ 底部导航栏：三个 Tab，当前激活"情绪趋势"

【视觉调性】
- 延续深色赛博朋克风格
- 图表用渐变色彩（蓝色→紫色→粉色）
- 卡片有轻微阴影和圆角
- 数据可视化清晰易读
```

## 设计使用技巧

### 1. 分阶段生成
- 先完成首页，确认风格后再生成其他页面
- 避免一次性生成所有页面

### 2. 调整提示词
如果生成的设计不符合预期，可以修改提示词：
- "增加更多赛博朋克元素"
- "天空效果更动感"
- "情绪选择器改成横向滑动"
- "城市天际线更突出"

### 3. 使用"规范出图"
- 所有页面生成完毕后使用
- 整理设计规范，包括颜色方案、字体规范、组件库

### 4. 设计还原
- 如果有参考图，可以使用"设计还原"功能
- 上传 UI 参考图，让 TRAE 还原设计
